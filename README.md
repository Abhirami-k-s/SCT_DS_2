## Objective
To perform data cleaning and exploratory data analysis on the Titanic dataset to understand factors affecting passenger survival.

## Dataset
- Source: Kaggle Titanic Dataset
- Rows: 891
- Columns: 12

## Steps Performed
- Handled missing values (Age, Embarked)
- Dropped irrelevant columns
- Feature analysis using barplots,boxplots
- Correlation analysis

## Key Insights
- Women had higher survival rates
- Higher class passengers survived more
- Fare positively influenced survival
